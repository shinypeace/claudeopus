# 🚀 Cosmic Shooter 3D

Полноценный 3D космический шутер, созданный с использованием Three.js. Сражайтесь с врагами на далекой планете, собирайте бонусы и побейте свой рекорд!

## 🎮 Особенности игры

- **Полноценный 3D геймплей** - FPS управление с мышью и клавиатурой
- **Разнообразные враги** - 4 типа врагов с уникальным ИИ и способностями
- **Система оружия** - 3 типа оружия с разными характеристиками
- **Бонусы и бусты** - Здоровье, патроны, урон и скорость
- **Красочная графика** - Динамическое звездное небо, туманности, планета
- **Процедурная музыка** - Атмосферная космическая музыка и звуковые эффекты
- **Система волн** - Усложняющиеся волны врагов
- **Статистика** - Отслеживание рекордов и достижений

## 🎯 Управление

- **WASD** - Движение
- **Мышь** - Поворот камеры
- **ЛКМ** - Стрельба
- **Пробел** - Прыжок
- **Shift** - Бег
- **R** - Перезарядка
- **ESC** - Пауза

- **Three.js** - 3D графика и рендеринг

- **Local Storage** - Сохранение статистики
- **CSS3** - Стилизация интерфейса

## 🎨 Игровые элементы

### Типы врагов:
1. **Базовый** (красный) - Стандартный враг с шипами
2. **Быстрый** (зеленый) - Быстро движется зигзагом
3. **Тяжелый** (фиолетовый) - Много здоровья, медленный
4. **Босс** (оранжевый) - Большой враг с особыми атаками

### Типы оружия:
1. **Штурмовая винтовка** - Сбалансированное оружие
2. **Плазменный дробовик** - Мощный на близкой дистанции
3. **Плазменная пушка** - Высокий урон, медленная стрельба

### Типы бонусов:
1. **Здоровье** (зеленый) - Восстанавливает HP
2. **Патроны** (желтый) - Пополняет боеприпасы
3. **Урон** (красный) - Увеличивает урон на 10 секунд
4. **Скорость** (синий) - Увеличивает скорость на 5 секунд

## 🌟 Визуальные эффекты

- Динамическое звездное небо с несколькими слоями
- Анимированные туманности
- Частицы от джетпака игрока
- Взрывы при уничтожении врагов
- Следы от плазменных пуль
- Атмосферные эффекты (молнии, метеоры)
- Пульсирующие бонусы с частицами

## 🎵 Аудио

Все звуки генерируются процедурно с использованием Web Audio API:
- Звуки выстрелов с вариациями
- Взрывы и попадания
- Сбор бонусов
- Атмосферная космическая музыка
- 3D позиционный звук

## 🚀 Запуск

1. Откройте `index.html` в современном браузере
2. Разрешите использование мыши для управления
3. Наслаждайтесь игрой!

### Для GitHub Pages:
Просто загрузите все файлы в репозиторий и включите GitHub Pages. Игра будет доступна по адресу: `https://ваш-username.github.io/имя-репозитория/`
